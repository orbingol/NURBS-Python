NURBS-Python Object-oriented NURBS library for Python

NURBS-Python (geomdl) is a pure Python object-oriented library containing implementations of NURBS curve and surface generation and evaluation algorithms.

Some significant features of NURBS-Python:

  • Self-contained, object-oriented, customizable API
  • Convenient and easy-to-use data structures for storing curve and surface descriptions
  • Knot vector and surface grid generators
  • On-the-fly visualization with Matplotlib and Plotly modules
  • Exporting curve and surface data into various file formats
  • Examples repository containing over 40 scripts describing usage and integration scenarios
  • A complete documentation with examples and illustrations
  • Tested with Python versions 2.7.x and 3.3+
  • Pure Python; all implemented in Python, no external C/C++ or FORTRAN library dependencies
  • Convenient installation options: PyPI and Conda
